# #47: Spice Key

A high-speed visual dictionary for 50+ essential spices. Designed for quick decision-making at supermarkets or in the kitchen.

## 1 Tool / 1 Action
- **Instant Filtering**: Search by name, alias, or usage (e.g., searching "Cilantro" will find "Coriander").
- **Visual Color Coding**: Each spice card features a color-coded border, allowing for intuitive identification of spice types.
- **Quick Clear**: A dedicated "clr" button to reset your search instantly.

## Core Philosophy
- **Performance**: Zero external database calls. All data is embedded within a single HTML file for lightning-fast results even offline.
- **Privacy**: No login / No tracking / No ads / Browser only.
- **Design**: Fridge Combo Signature UI (White background, #333 elements, 15px border-radius).

## Global Friendly
- Supports multiple names for the same spice (e.g., American vs. British English, or local Malay names like "Kunyit" for Turmeric).
- Minimalist 3-word usage guides for immediate understanding.
